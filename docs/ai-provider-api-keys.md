# AI Provider API Keys

CF7 Mate's AI Form Generator supports 4 providers. You only need **one** API key to get started.

---

## OpenAI

1. Go to [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Sign in or create an account
3. Click **"Create new secret key"**
4. Give it a name (e.g. "CF7 Mate") and click **Create**
5. Copy the key (starts with `sk-`)
6. Add billing: Go to **Settings > Billing** and add a payment method

**Recommended model:** GPT-4o Mini

---

## Anthropic (Claude)

1. Go to [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys)
2. Sign in or create an account
3. Click **"Create Key"**
4. Name it (e.g. "CF7 Mate") and click **Create Key**
5. Copy the key (starts with `sk-ant-`)
6. Add billing: Go to **Settings > Billing** and add a payment method

**Recommended model:** Claude Sonnet 4

---

## Grok (xAI)

1. Go to [console.x.ai](https://console.x.ai/)
2. Sign in with your X (Twitter) account
3. Navigate to **API Keys**
4. Click **"Create API Key"**
5. Copy the key (starts with `xai-`)
6. Add credits under the **Billing** section

**Recommended model:** Grok 3 Mini

---

## Kimi (Moonshot)

1. Go to [platform.moonshot.cn/console/api-keys](https://platform.moonshot.cn/console/api-keys)
2. Sign in or create an account
3. Click **"Create API Key"**
4. Copy the key (starts with `sk-`)
5. Top up credits under **Billing**

**Recommended model:** Moonshot 32K

---

## Using the Key in CF7 Mate

1. In WordPress admin, go to **CF7 Mate > AI Provider**
2. Select your provider
3. Paste the API key
4. Click **Test Connection** to verify
5. Save settings

Your API key is encrypted before storage and never exposed in the admin UI.
