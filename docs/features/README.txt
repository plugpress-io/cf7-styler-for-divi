CF7 Mate – Feature docs (plain text)
=====================================

For an overview of all 19 features, how they work, and how they help users, see:
  ../FEATURES_GUIDE.md

Each file below describes one feature: what it is and how to use it. Short and simple.

  star-rating.txt                – Star rating field
  range-slider.txt               – Range slider field
  separator.txt                  – Horizontal divider between sections
  image-field.txt                – Image inside the form
  icon-field.txt                 – Icon inside the form

  form-entries.txt               – Save submissions, view and export in Entries
  phone-number-field.txt         – Phone input with country selector

  calculator.txt                 – Live calculator / price estimator
  conditional-logic.txt          – Show/hide fields by other field values
  ai-form-generator.txt         – Generate forms with AI
  presets.txt                    – One-click design presets
  webhook.txt                    – Send submissions to external URLs
