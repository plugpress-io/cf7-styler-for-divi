Heading Field (Pro)
===================

What it is
  Inserts a heading (H1–H6) inside the form to label sections (e.g. "Your details", "Additional info"). Display only; no data is submitted.

How to use
  1. Enable "Heading Field" in CF7 Mate → Modules (Pro).
  2. Edit your Contact Form 7 form. Use the CF7 Mate Heading tag generator: choose level (H1–H6) and type the heading text. Insert the tag.
  3. Save the form.
  4. Place the form with the CF7 Styler module. Style headings in the module’s design (font size, color, spacing).

Use it to break long forms into clear sections and improve accessibility.
