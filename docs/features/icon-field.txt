Icon Field
==========

What it is
  Adds an icon (e.g. checkmark, info, warning) next to or above form content. Display only; no data is submitted.

How to use
  1. Edit your Contact Form 7 form (Contact → Contact Forms).
  2. Open the CF7 Mate Icon tag generator, pick an icon and optional size/position. Insert the tag into the form.
  3. Save the form.
  4. Add the form to a page with the CF7 Styler module. Style the icon in the module’s design options if needed.

Good for highlighting important notes or steps in the form.
